## Description

Repository that aggregates knowledge about [NestJS](https://github.com/nestjs/nest) framework

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
